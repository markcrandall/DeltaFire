# DeltaFire

Addition and subtraction math game for 8-year-olds. Defend your tower from space alien invaders by solving math problems!

A sibling game to [MultiFire](../MultiFire/) — same tower-defense gameplay, different math operations.

## Difficulty Tiers

| Tier | Streak | Problems |
|------|--------|----------|
| Bullet | 0-4 | Single + single, single - single |
| Fireball | 5-9 | Double + single, double - single |
| Lightning | 10-19 | Double + double, double - double (10-50) |
| Laser | 20+ | Double + double, double - double (10-99) |

## Play

Open `index.html` in any modern browser.
